# Piwork Go-To-Market Playbook

## Phase 1: Bootstrap (Days 1-14) - Goal: 5 Customers + 10 Freelancers

### Customer Acquisition Script

**Target:** People posting "looking for [service]" in Facebook groups

**Process:**
1. Search Facebook groups: "logo design Philippines", "hire designer", "virtual assistant"
2. Filter: Posts from past 7 days containing "looking for", "need", "seeking"
3. Personal message with hook: "I built a platform where you hire designers with ZERO bank fees"
4. Offer: "First task is free commission - want a quick demo?"
5. Demo: 15-minute screen share showing:
   - Task creation (30s)
   - Freelancer browsing (1m)
   - Payment escrow explanation (2m)
   - Real deal walkthrough (5m)
6. Close: "Ready to create your first task?"

**Success Metric:** Get them to post first 20 Pi budget task within 24 hours

**Template Messages:**
```
Hi {{name}},

I saw you're looking for {{service}}. Just launched PiWork - hire freelancers and pay in Pi with NO bank fees.

First task is free commission (you save 2%). Want to see it in action? Takes 5 min.

[Your phone number or Zoom link]
```

---

### Freelancer Acquisition Script

**Target:** Comments in Facebook groups + DM follow-up

**Process:**
1. Post in "work from home Philippines", "online jobs Nigeria" groups:
   ```
   🚀 Earn Pi with your skills - No experience needed!
   
   Hiring: Logo Design, Writing, Data Entry, Voice Recording
   ✓ Payment guaranteed in escrow
   ✓ Keep 98% (2% fee only)
   ✓ Instant Pi payout
   
   Comment your skill + Pi username for test task (10 Pi paid!)
   ```

2. Respond to comments with instructions
3. Send personalized DM: "Hi {{name}}, thanks for commenting! I'm hiring {{skill}} freelancers. Here's a test task (10 Pi) to try the platform."
4. Provide walkthrough video for first task
5. After completion: "Nice work! Here are 3 paid tasks (50-100 Pi each)"

**Template:**
```
Hi {{name}},

Thanks for commenting! Test task incoming:
- Task: {{description}}
- Pay: 10 Pi (guaranteed in escrow)
- Time: 30-60 minutes
- Support: I'll help you through it

Check your inbox in PiWork - start anytime!
```

---

## Metrics to Track

- **Customers:** DMs sent → Demos scheduled → Demos completed → First task posted
- **Freelancers:** Comments received → DMs sent → Test task started → Test task completed → Paid task accepted
- **Conversion:** Daily/weekly rates, cost per customer/freelancer acquired

## Week 1 Targets
- 10-15 customer DMs sent
- 5 demos scheduled
- 2-3 first tasks posted
- 20-30 freelancer group posts
- 50+ comments received
- 10 test tasks assigned

## Sustainability
Once initial cohort is active, they become:
1. Referral sources (ambassador program)
2. Case studies for organic growth
3. Revenue engine for team salary
